## J-Link 一键烧写脚本使用指南

此脚本会自动将名为 `LHD8006_bootloader.bin` 的固件烧写到 0x08000000 的位置。再将名为 `LHD8006_application.bin` 的固件烧写到 0x08008000 的位置。

### 流程

- 1、复制编译生成的最新 bin 文件到此文件夹中，修改文件名为：`LHD8006_application.bin` 或 `LHD8006_bootloader.bin`；
- 2、连接好仿真器与待下载电路板；
- 3、点击 `点我烧写.bat` 完成烧写，如果烧写过程中出现卡死或非正常结束的现象，请检查电路板与仿真器之间的连接。
